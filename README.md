# Perpustakaan Digital

Aplikasi perpustakaan digital berbasis mobile menggunakan React Native Expo dan Firebase.

## Fitur
- Login & Register (Firebase Auth)
- Peminjaman Buku
- Pengembalian Buku
- Riwayat Peminjaman

## Cara Menjalankan
1. Clone repository ini.
2. Install dependencies:
   npm install
3. Jalankan aplikasi:
   npx expo start
